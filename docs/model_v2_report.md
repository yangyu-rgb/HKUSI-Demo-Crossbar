# AI v2 合成场景模型评估

> 本模型用于课堂场景演示，所有标签均由可解释、确定性的场景公式生成，不代表真实口岸预测准确率。

## 数据与模型

- 模型：`synthetic-scenario-hgb-v2`（HistGradientBoostingRegressor）
- 样本：43,008 条，覆盖四口岸、双向、224 天和逐小时切片
- 特征：口岸、方向、周期时间、星期、周末、节假日、四类天气和事件强度
- 切分：训练截至 2025-06-05，验证截至 2025-07-09，测试截至 2025-08-12

## 时间切分结果

| 分组 | 样本数 | MAE | RMSE |
| --- | ---: | ---: | ---: |
| 验证 | 6,528 | 1.4938 | 1.7826 |
| 测试 | 6,528 | 1.5001 | 1.7814 |

测试集的口岸/方向/小时日历基线 MAE 为 4.3511 分钟，V2 在同一测试集上改善约 65.5%。分场景 MAE 为：晴朗 1.2862、降雨 1.3931、暴雨 1.6378、雷暴 1.6670；无事件到高强度事件的 MAE 范围为 1.2956–1.7378。

四口岸验证残差的 90% 分位数为 2.72–2.85 分钟，用于用户可见置信区间。模型直接预测等待时间；确定性路线优化层再结合交通矩阵、预算、偏好和到达时间生成推荐。

## 运行边界

- 预测响应明确返回 `prediction_engine=v2` 和场景快照。
- 产物缺失、数据哈希不匹配或推理失败时自动降级到统计模型。
- 合成指标只能证明场景响应和工程流程可复现，不能用于真实效果、商业准确率或生产可靠性声明。
