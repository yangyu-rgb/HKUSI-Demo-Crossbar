## 限制与后续

- 数据由确定性公式生成，模型可能只是在学习生成规则。
- 天气、节假日和口岸分布不代表真实世界频率。
- FastAPI 仅在影子模式加载通过校验的模型，并记录其与统计模型的差异；不会改变当前产品行为或用户结果。
- 重复事件当前只作用于统计预测器；AI v1 没有对应事件特征，因此影子差异会包含该运行时调整。
- 获得真实数据后必须重新定义时间范围，并重新划分训练、验证和测试集。
- 合成数据指标不得进入正式效果声明或商业材料。
