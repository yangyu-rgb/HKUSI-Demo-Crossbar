## How to Add Screenshots

1. Take screenshots at 1440×900 resolution for consistency
2. Save as PNG format
3. Optimize file size (use tools like TinyPNG, ImageOptim)
4. Name files descriptively (lowercase, hyphenated)
5. Update README.md image links

## Example README Syntax

```markdown
![Homepage - Border Situation](screenshots/homepage.png)
```

## Guidelines

- Use light mode for better readability
- Capture complete page layouts (avoid cropping important UI)
- Include realistic demo data (not empty states)
- Show the platform in English (system language)
- Avoid sensitive or test data in screenshots
