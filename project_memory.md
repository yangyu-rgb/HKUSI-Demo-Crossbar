## 已确定决策

- CrossBorder AI 是可执行代码、样例数据和辅助文档中的唯一产品主题。
- 原型保持确定性并仅用于 Demo，不依赖真实 i口岸、地图、交通、天气、通知或外部 AI 服务。
- Demo 预测器使用时间匹配历史基线和最大 15% 的衰减众包修正，并结合历史波动率、预测斜率不确定性和概率风险；生产模型以后可以在不改变主要 API 边界的情况下替换该服务。
- 反馈、订阅和企业方案历史保存在本地 SQLite；模拟元数据保存在 JSON；显式重置接口按当时香港时间恢复相对时间种子。
- 预测 API 边界使用稳定地点 ID，`/api/predict` 不接受自由文本地点。
- 根据香港当前时间计算预计到达时间，并在最晚出发建议中加入十分钟安全缓冲。
- TanStack Query 负责共享服务端状态、轮询、Mutation 和缓存失效；临时表单和编辑状态保留在功能页面内。
- 前端契约类型由后端 OpenAPI 文档生成，不重复手写响应接口。
- B2B SaaS 面向跨境巴士、物流和企业 HR，是主要商业方向；个人流程用于展示产品价值和数据闭环。
- `AGENTS.md` 保存仓库操作规则；根目录 `project_memory.md` 保存任务历史、验证、同步说明和后续事项。
- 仓库自有说明文档使用中文；框架名、命令、路径、API 名称和字段等专有名词保留原文。
- 项目记忆仅记录日期、任务、修改范围、验证结果和必要的同步说明。
- 众包反馈质量由新鲜度、等待偏差和人流一致性确定；原始记录保留在 SQLite，过期数据不显示也不参与预测。
- 本地开发默认从仓库根目录运行 `./start.sh`；脚本缺少依赖时自动初始化，独立服务命令仅用于调试。
- Agent 可以执行必要的 Git 只读检查和上游刷新，但不执行 `git add`、`git commit` 或 `git push`；任务结束时只提供建议的 commit 文案。
- 只有平台时钟是真实数据；口岸、天气、交通和事件均明确为模拟输入，不调用外部服务。
